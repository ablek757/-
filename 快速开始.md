# 项目说明

## 这是什么？

这是一个对比**Prompt工程**和**模型微调**在智能客服意图识别任务上的完整实验项目。

## 核心价值

帮助你决策：什么时候用Prompt够了，什么时候必须微调。

## 3分钟快速上手

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 运行分析
python scripts/comprehensive_analysis.py
python scripts/generate_final_report.py

# 3. 查看报告
cat results/final_decision_report.md
```

## 核心结论

- **准确率**: Few-shot Prompt (90.0%) ≈ 微调 (89.5%)
- **成本**: 小规模用Prompt，大规模用微调（成本差60-170倍）
- **延迟**: 微调快2.7倍（300ms vs 800ms）
- **避坑**: CoT推理在本任务中适得其反（准确率降至81.5%）

## 适合谁？

- 做客服系统的产品经理/技术负责人
- 研究Prompt工程 vs 微调的NLP工程师
- 需要意图识别方案决策的团队

## 数据集

1000条电商客服对话，15类意图，已标注。

## 技术栈

- Prompt: 阿里云DashScope Qwen-Plus
- 微调: 阿里云百炼 Qwen3-4B LoRA
- 评估: scikit-learn

## 文件导航

- `results/final_decision_report.md` - 完整分析报告（必读）
- `intent_taxonomy.md` - 15类意图定义
- `prompts/` - 4种Prompt策略
- `scripts/comprehensive_analysis.py` - 评估框架核心代码
